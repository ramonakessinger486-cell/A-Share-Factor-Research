# Title: A-Share Micro-liquidity & Fama-French Factor Research
# Project Overview: 本项目使用 Python 调用 CSMAR 接口抓取近 10 年 A 股交易数据，通过 Fama-French 三因子模型，实证研究了传统价值因子在特定宏观周期内的失效性。
# Key Skills: Python (Pandas/NumPy), Data Cleaning, Quantitative Finance.
Structure:
01_data_cleaning.ipynb: 异常值清洗与 MAD 缩尾处理逻辑。
02_factor_construction.ipynb: 因子构建与回归分析。
